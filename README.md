# RAG Chatbot System

## Overview

This project demonstrates the implementation of Retrieval-Augmented Generation (RAG) chatbots using a custom knowledge base. The system retrieves relevant information from vector databases and provides accurate responses based on the available knowledge.

## Project Objectives

### 1. IPSR Courses Chatbot

An AI-powered chatbot designed to recommend IPSR courses based on user interests, learning goals, and available course information.

### 2. Union Budget Chatbot

A chatbot that answers budget-related questions using a dedicated Union Budget knowledge base and retrieval system.

## Technologies Used

* Python
* Retrieval-Augmented Generation (RAG)
* Hugging Face Embeddings
* Pinecone Vector Database
* n8n Workflow Automation
* Large Language Models (LLMs)

## System Workflow

1. Collect knowledge base documents.
2. Split documents into smaller text chunks.
3. Generate vector embeddings using Hugging Face.
4. Store embeddings in Pinecone.
5. Retrieve relevant information based on user queries.
6. Generate accurate responses using RAG architecture.

## Features

* Intelligent document retrieval
* Context-aware responses
* AI-powered course recommendations
* Budget information assistance
* Vector database integration
* Scalable knowledge base architecture

## Knowledge Bases

### IPSR Courses Knowledge Base

Provides information about available IPSR courses and helps users select suitable learning paths.

### Union Budget Knowledge Base

Provides explanations and answers related to Union Budget topics and financial information.

## Project Outcomes

* Successfully built custom knowledge bases.
* Generated embeddings using Hugging Face.
* Stored and managed vectors in Pinecone.
* Implemented RAG-based chatbot workflows.
* Delivered accurate responses using retrieved context.
